# Менің бірінші ЖИ-ассистентім

## Жоба туралы
Бұл веб-бет ЖИ-құралдарын (GitHub Copilot, ChatGPT, Claude) көрсетеді, олардың сипаттамасын береді және нақты OpenAI API арқылы жұмыс істейтін чат бар.

## Қалай іске қосу керек
1. `index.html` файлын браузерде ашамыз.
2. Веб-бетте ЖИ-құралдарын бойынша ақпарат алып, чатта кез келген сұрақ қою арқылы жауап ала аласыз .
3. Барлығы дайын!

## Пайдаланылған құралдар
- HTML5 + CSS3 (Flex/Grid)
- Vanilla JavaScript
- OpenAI API (чат)
- ChatGPT + Claude (мәтін және код генерациясы үшін)

## Мен не үйрендім
- ЖИ API-ін браузерден тікелей қосу
- Responsive дизайн және CSS айнымалыларын
- Intersection Observer + Dark Mode
- Кодтың әр жолын түсіну біз маңызды екенін
